# HTML 模板资产说明

`advisor-report-template.html` 是导师阅读版的默认模板，版式来自已经多轮修改的当前周报。

生成新报告时：

1. 复制该 HTML，不从空白页面重写。
2. 替换所有 `{{...}}` 占位符。
3. 按用户实际工作替换章节文案、表格、Mermaid 节点、ECharts 数据和 D3 时间数据。
4. 根据当前阶段调整节点的 `done/current` 状态。
5. 保留图表的文字降级说明，避免 CDN 失败时报告无法阅读。
6. 交付前检查 HTML、内联 JavaScript、旧日期、旧人名和未替换占位符。

只有用户明确要求改版时，才改变整体视觉结构。
